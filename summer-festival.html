Summer Festival

Design Background

Running has become very popular in recent years, so there are so many marathon races. This project is a running event that holds in the summer. Through this event, Garmin users can use their watches to join and participate the T-shirt drawing.

Role and Collaborations

My role in this project was to design the layout, web animation and coding. I coordinated with the project manager to discuss how to let users quickly know the rule and register.

Design Goal

Emotional design

Through the page design that user can enhance the motivation to join the event or buy watches.

Colorful and clear

Use colorful design to present more running power and show the clear event information.

Interaction

Use animation to attract users to join the event.

Outcome and Impact

More than expected runners have join the event. Runners are interesting of the event.

The sale of the promotion watches is good.

From the GA data, the page view and stay time is good.

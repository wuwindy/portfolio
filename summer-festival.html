Summer Festival

Design Goal: Create an event landing page to let users know the rule and register the campaign of the summer festival


Character

My role in this project was to design the layout, web animation and coding. I coordinated with the project manager to discuss how to let users quickly know the rule and register.

Design

In order to responsive summer energy, used green as the main color, and used yellow as the highlight color. And using parallax to make the wave animated let the landing page more attractive. The landing page sings a responsive design that supports different form factors and devices.

Results

Many users have participated in the summer running event. It looks to achieve the goal of the running campaign.

Summer Festival

Design Background

Running has become very popular in recent years, so there are so many marathon races. This project is a running event that holds in the summer. Through this event, Garmin users can use their watches to join and participate the T-shirt drawing.

Design Goal

01

Emotional design

Through the page design that user can enhance the motivation to join the event or buy watches

02

Colorful and clear

Use colorful design to present more running power and show the clear event information

03

Interaction

Use animation to attract users to join the event

User Research

Age:30

Situation - I already have Garmin watches, so I can run and upload the data to Garmin Connect to join the event

Team Character

My role in this project was to design the layout, web animation and coding. I coordinated with the project manager to discuss how to let users quickly know the rule and register.

Design

In order to responsive summer energy, used green as the main color, and used yellow as the highlight color. And using parallax to make the wave animated let the landing page more attractive. The landing page sings a responsive design that supports different form factors and devices.

Results

Many users have participated in the summer running event. Over 300 users have participated in the T-shirt drawing. And also successful at the promotion of the Garmin running watches.

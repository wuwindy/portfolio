Garmin Sports APP Design

Design Goal:Create the race competition layout to let users join the runner race



Challenge: The layout suffers conflicting visual elements and race information

We realized that provide a clear race score is the most important. So we hide some information which isn't important during the race.

Plan & Solution

We understood the challenges, and start the project from realize our users and define the flow.

UI design

Identify the best UI strategy and placement. We explored multiple design options to address three essential questions:

  • 1. Which option is more visibility ?
  • 2. Which option is simpler?
  • 3. Which option is more user friendly?

Garmin Sports APP Design

Design Background

As we know running and workouts can keep you in shape. This project is an event that holds in the APAC region. To attract more Garmin app users to run, hold the event to attract the user who wants to lose weight. And hope through the running event can let the user love running.

Design Goal

01

Emotional design

Do the design of the awards to let users more engage in the event

02

Know the event information

According to the three steps of the events, use color to define the hierarchy and let user know the current important information

03

Motivation

Through the event to let users enhance the running habits

User Research

Pain point - want to run but doesn't have the motivation

Requirement - Make running become everyday event

Design

Challenge: The layout suffers conflicting visual elements and race information

We realized that provide a clear race score is the most important. So we hide some information which isn't important during the race.

Results

Planner and PM feedback that the event holds successfully. And many runners feedback is positive and after the event would also want to run.
